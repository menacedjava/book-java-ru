Урок №1: Что такое Java?

Здравствуйте! На сегодняшнем уроке мы узнаем, что такое язык программирования Java.

Java — это объектно-ориентированный, независимый от платформы язык программирования. Самая важная особенность, которая отличает Java от других языков, заключается в принципе: "один раз написал — работает везде". Это означает, что программу, написанную на Java, можно запускать на различных операционных системах - таких, как Windows, Mac, Linux — без каких-либо изменений.

История создания Java уходит в 1995 год. Язык был разработан компанией Sun Microsystems, а позже куплен компанией Oracle, которая продолжает развивать Java по сей день.

С помощью Java можно создавать мобильные приложения, веб-сайты, серверные приложения, игры, а также крупные корпоративные системы. Благодаря своей удобности и безопасности, Java используется миллионами разработчиков по всему миру.

#Вывод

В заключении можно сказать, что Java - это современный и мощный язык программирования, предназначенный для разработки широкого спектра приложений. Он отличается своей независимостью от платформы, что делает его особенно удобным и универсальным

Урок №2 Понятие JDK, JRE and JVM

Что такое JVM?

JVM - это среда выполнения Java-программ. То есть, Java-программа компилируется в байт-код, и именно JVM понимает и выполняет этот байт-код. JVM позволяет запускать Java-программы на разных платформах(например, Windows, или Linux), так как она адаптирована под каждую платформу.

Имейте в виду, что JVM не выполняет Java-код напрямую, а работает с кодом, который предварительно был преобразован в байт-код.

2.Что такое JRE(Java Runtime Environment)?

JRE-это набор компонентов, необходимых для выполнения Java-программ. В составе JRE находится JVM, а также библиотеки Java и необходимые файлы. Если вы не планируете писать Java-программы, а запускать уже готовые, вам достаточно будет JRE.

Таким образом, JRE-это комплект предоставляющий всё необходимое для выполнения Java- программ.

3. Что такое JDK(Java Development Kit)?

JDK-это набор компонентов, необходимых для создания и разработки Java-программ. В JDK входят JRE и JVM, а также компилятор, отладчики и другие вспомогательные утилиты для разработки программ

Если вы собираетесь писать Java-программы, вам нужен JDK. Он позволяет не только выполнять Java-программы, но и компилировать и разрабатывать их.

#Вывод

Подводя итоги: JVM-виртуальная машина, которая выполняет Java- программу. JRE — среда,необходимая для запуска
